
cd name-перемешает в ккакую то папку(директорию)
ls -просматривает содержимое дирекктории
touch name  -сосдает файл
mkdir name- создает папку(директорию)   
ls -a -показывает файлы вместе со скрытыми файлами и папками
ls -l -показывает файлы и папкки с доп инфрмацией
ls -la -показывает файлы и папки со скрытими файлами +доп информации

cd - переходит в домашнуб папку
cd ~ -переходит в домашную папку 
cd . -переходит в ту папку в которой вы нахордитесь
cd .. -переходит в предыдущую папкку (родительская папка)
rm name -удаление файла name 
rmdir -удвление пустой директории ?(папки)
rm -rf -удвление все /папки с содержимым /файлы/пустые папки
history -выводит историю комаанд
history какое то число -выводит историю в каком то кколисчестве
history | grep слово -выводит все команды /в которвых есть указанное слово

pwd -показывает абсолютный(начиная с корневой папки) путь до текущей территории в которой вы находитесь
cat -выводит содержмимое файла
nano -текстовый редактор /при помощи которого мы можем записать данные в файл



mv test1.txt test3.txt -переименовывает файл test1.txt v test3.txt
mv test1 test3 -переименоввает папку test1 v test3
mv test1.txt test1 -перемешает файл test1.txt в папку test1
apt -системный установщик
snap -установщик пакетов (чаще всего используют для установки программ)
sudo (superuserdo)-выполнение какой либо команды от имени администратора)
